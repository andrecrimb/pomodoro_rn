# Time tracker - Pomodoro App ![example workflow](https://github.com/andrecrimb/pomodoro_rn/actions/workflows/main.yml/badge.svg)

React native app that helps you to track your time and increase your productivity using the Pomodoro method.

### Main tech/framework used
- [React Native (Expo managed)](https://docs.expo.dev/get-started/installation/)
- [XState](https://xstate.js.org/docs/guides/start.html#our-first-machine) 
- [React Navigation](https://reactnavigation.org/)
- [Emotion](https://emotion.sh/docs/introduction)
- [React Native Reanimated 2](https://github.com/software-mansion/react-native-reanimated#readme)

### Cool Features
- Flatlist swipe to delete using Reanimated 2, React Native Gesture Handler and Expo haptics
- XState state machine managing time sections

### Requirements
Make sure to have installed:  
-   [Node.js LTS release](https://nodejs.org/en/)
-   [Git](https://git-scm.com/)
-   [Watchman](https://facebook.github.io/watchman/docs/install#buildinstall)  for macOS or Linux users
- [Expo](https://docs.expo.dev/get-started/installation/)

### Tests
To run all tests you just need to run:
		
	 yarn test
